# myukkurivoice
myukkurivoiceは、AquesTalkを利用したMac OSX向けの  
動画制作向け合成音声生成アプリケーションです。  
macOS Sierra 以降に対応。

## 開発ステータス
* 開発中
* AquesTalk評価版使用中

## ダウンロード
* https://github.com/taku-o/myukkurivoice/releases

## 主な特徴
* MacOSX Sierra以降で動作検証。
* 設定を大量に用意して切り替えて作業する
* 動画制作向きのショートカットキー定義
* 音声ファイルを連番付きの名前で保存(使用未使用選択可)
* 音声ファイル元メッセージ保存機能(使用未使用選択可)
* チュートリアル
